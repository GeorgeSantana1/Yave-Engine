# Yave
Yet another C++14 Vulkan engine.

## Building
You need:
 * Cmake 3.6
 * gcc 5
 * [Vulkan SDK](https://lunarg.com/vulkan-sdk/)
 * [Vulkan hpp](https://github.com/KhronosGroup/Vulkan-Hpp)
 * [Spriv_cross](https://github.com/KhronosGroup/SPIRV-Cross)
 * [y](https://github.com/gan74/y)
